Google

Description

When programming basic intelligence you may want to search the internet to gather information. The Google module uses the PyGoogle interface, which is a wrapper for the Google Web APIs. Pygoogle is bundled in the download below, however, you need to obtain a Google license key to be able to use the Google Web APIs. Follow the steps at http://www.google.com/apis/ to set up a Google Account and obtain a license key.

Google has ceased support for their API. Read more»
