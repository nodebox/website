Organic Universe

A piece generated for Ad!dict design magazine. It's basically the same code as used for Geospheres, an example script included in the Core Image download, but with extra connections between different spheres and some fine-tuning.

For some reason when I was making this I recalled Jef Bertels' work - his crazy paintings are some of the earliest I encountered in my childhood. They still look as wonderful as they did back then.

Right now I'm looking for a way to do stereographic projections onto my furry planets.
