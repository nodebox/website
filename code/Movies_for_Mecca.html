Movies for Mecca

The guys at Mecca have been testing NodeBox's animation capabilities, resulting in some wonderful trailers for their site. The NodeBox scripts for the movies use a simple sine function to move an image (or two images, the front and the sky) across a fluid camera path). Click the images and the movies will load in a pop-up window (please be patient).
