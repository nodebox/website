Experimental type posters

Guy Haviv created a generative poster series displaying all typefaces installed on his Mac. He utilized Apple's Typography API using PyObjC. Each poster shows the typeface names, along with circles proportional to their weights. The names are sorted, rotated & resized according to various parameters of the typeface, such as italic angle, weight, etc.
