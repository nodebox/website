Reference | rect()


Syntax
rect(x, y, width, height, roundness=0.0, draw=True)

DescriptionDraws a rectangle to the screen. The first two parameters set the location (measured from the rectangle's top-left corner), the third sets the width, and the fourth sets the height. An optional fifth parameter controls the corner roundness of the rectangle.
ReturnsBezierPath containing the rectangle
TutorialPrimitives, Paths


Example
fill(0.2)
rect(10, 20, 60, 40)
