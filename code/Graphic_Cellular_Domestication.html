Graphic Cellular Domestication

As part of her postgraduate Transmedia program in graphic design Ludivine Lechat worked with genetic recombination. Starting out with a set of parental cells and evolutionary rules (based on colonies, virii and seaweed) she created a hive of evolving creatures.

These are some experiments in which NodeBox is populated with these entities. In the movie you'll see two of them initiating in some kind of bizarre mating ritual. Both creatures have a goal-oriented behaviour - they orbit around a defined point in space. Is this case, that point is each other.

A next step is fleeing, eating, multiplying along with a set of genetic code that lets the creatures evolve by themselves (these were handcrafted) - to be continued!
