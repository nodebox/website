Frederic Albers | Fairytales

Frédéric Albers is currently finishing up on his postgraduate program in graphic design at the St. Lucas School of Arts in Antwerp. Frédéric used the Core Image library to create an engine that visualises fairytales and dreams. The engine uses images it fetches live off the internet.
